Add a parameter by INVOKING INPUTS!

in home.html...

  < a ui-sref="inputs()">INPUTS< / a>

Then, we put in an object.

< a ui-sref="inputs({})">INPUTS< / a>

And this Object will be the $stateParams

NB: We can only give it params that we have specified in the url path.
It can only accept $stateParams that we've set in our app.js file.

Thus:
< a ui-sref="inputs({id: 'default input param'})">INPUTS< / a>

This is ui-router's way of passing different states in your code.
